Cookies Policy

What is a cookie?

A cookie is information sent by a website and stored on the user's browser to know about previous user activity.

Types of cookies

According to their purpose:

  • Functional or technical: allows the user to use an application or web page with all its functions, such as those that offer customization possibilities..
  • Analytics: allows the analysis of user behavior and actions within the application or website. This information is used to create user browsing profiles, which are used to obtain statistical information and improve the operation of the website.
  • Advertising: allow the management of advertising space within the application or website and record information about the ads displayed and user behavior with these.
  • Behavioral advertising: allow to analyze the user's preferences and personal choices within the website for external advertising platforms, such as in which ads are displayed in the application or website.

According to its duration:

  • Session: stored on the user's device while using the application or web page.
  • Persistent: stored on the user's device for a period of time defined by the cookie manager, which can last from days to years.

According to its owner:

  • Own: managed directly by the managers or owners of the application or website.
  • Third-party:: managed by an external provider.

Cookies settings

In the browser settings, the user can accept or reject the installation of cookies. Also, after each session you can delete all or some of the stored cookies.

The user can turn on private browsing in some browsers. With 'private browsing' your browser lets you save your browsing history, passwords, web sites, cookies and other information from the pages you visit.

We recommend you to consult your browser's help for options on how to manage cookies. Note that blocking cookies may affect some features of our website.

You can find more information on how disabling cookies in the help pages of the your browser:
